Welcome to Posties!

Posties is a unique social media application where users can interact through putting "sticky notes" onto a virtual board.
To create posts, you must first make an account and log in.

How to use the site:
Create a post by clicking on any empty space within a grid cell (login required).
Edit or delete a post by clicking on a post you made (login required).
View a user's profile by clicking on a post that is not your own.
