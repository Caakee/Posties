Welcome to Posties!

Posties is a unique social media application where users can interact through putting "sticky notes" onto a virtual board.
To use the site, you must first make an account and log in.

If you are logged in:
To create a post, click on any empty space within a grid cell and type a message.
To edit a post, left-click on any post you created and type a new message.
To delete a post, right-click on any post you created.
